Drawer Management System User Guide

This guide will help you navigate and use the Drawer Management System, a software tool designed to securely manage and organize various items into virtual drawers.

Features

  • Encrypted Storage: All your data is securely encrypted using a password-derived key.
  • Drawer Operations: Create new drawers, add or remove items, and delete drawers as needed.
  • Search Functionality: Quickly find items across all drawers.
  • Data Integrity: Data is saved upon exit and loaded at start-up, ensuring data persistence.
  • Interactive GUI: A simple and intuitive graphical user interface powered by PySimpleGUI.

Instructions

Starting the Application

Launch the program and enter your username and password. If no existing data is found for the username, a new database is created.

Adding a New Drawer

Click the New Drawer button, enter the name of the drawer and begin adding items by specifying their name and quantity.

Modifying a Drawer

To add or remove items, select the drawer and choose the appropriate action. You can specify the item and update the quantity or remove it completely.

Searching for Items

Use the search function to locate items across all drawers. The total quantity and specific drawer locations will be displayed.

Viewing Drawer Contents

Select a drawer to view its contents. Each item's name and quantity will be listed.

Exiting the Application

Click the Exit button to close the application. All changes will be saved securely.

Target Audience

This system is ideal for individuals or businesses needing a secure, simple solution to manage inventory or miscellaneous items.
